💡 Solution Overview
FraudShield AI has two complete portals in one unified platform:
🛡️ Portal 1 — Customer Fraud Detection
L1 — Synthetic ID Detector
Upload Aadhaar or PAN document image or type the ID number. Drag and drop supported. Scanning beam animates over your uploaded document. EfficientNet-B4 and ViT detect deepfake artifacts. Returns verdict with confidence score and SHAP breakdown. 8 scenario presets available.
L2 — Behavioral Anomaly Engine
Enter any account number or choose from 10 transaction presets. Isolation Forest and LSTM detect velocity anomalies and pass-through patterns. Live anomaly score chart. Pipeline context carries forward from L1.
L3 — Graph Network Intelligence
Dynamic fraud network graph generated per account using seeded PRNG. Star topology mule ring detection. Clickable nodes with inspector panel. GNN confidence score and topology classification.
L4 — Agentic STR and Evidence Generator
Real Claude API calls — watch the text type out live. Generate STR button creates full RBI-format Suspicious Transaction Report. Generate Evidence Package creates prosecution-ready court document. Both include SHAP explanations and PMLA legal sections. Human-in-the-loop review required.
L5 — Intel Network and Score API
Federated fraud intelligence network with live institution count. FraudShield Score API with JSON response and gauge display. Three pricing tiers: Starter, Pro (Rs.2/query), Enterprise.
🕵️ Portal 2 — Insider Threat Detection
Employee Risk Monitor
Search any Employee ID (U047, U089, U134, U201, EMP-8814) or type any random ID. Known employees show real forensic data with full SHAP breakdown. Unknown IDs get dynamically generated risk analysis. Hard Lock, Soft Lock, Clear as False Alarm buttons with confirmation messages. 10 employee scenario presets.
Collusion Graph
Louvain community detection visualizes Ring R001 (U047 + U089 + U134 confirmed) and Ring R002 (U201 suspected). Animated pulse on flagged nodes. Lock All Ring Members button.
Lockdown Control
Four score-based levels with visual threshold tiles. Manual override — enter any employee ID and apply any lock level with justification. Currently locked accounts panel with unlock and hard-lock escalation.
Alert Audit Trail
Immutable tamper-proof log of every action. Who flagged what, who cleared what, when and why. RBI EWS compliance status panel showing four compliance checks.
Evidence Package
Claude API generates insider fraud prosecution evidence live. Separate button for Insider STR report in RBI FIU-IND format. SHAP risk factors injected automatically into every document.
